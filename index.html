1자료 · 정보 · 데이터 분석이란?
자료자료 (Data)
관찰·측정으로 수집한 날것의 수치·사실. 그 자체로는 의미를 판단하기 어렵다.
예) 한국 기대수명 84.3세 / 출산율 0.72명
정보정보 (Information)
자료를 분석·가공하여 의미를 부여한 것.
예) "한국은 오래 살지만 아이를 가장 적게 낳는 나라다"
분석데이터 분석
자료에서 패턴·경향·관계를 찾아 유용한 정보를 만드는 과정.
예) 기대수명↑ → 출산율↓ 경향 발견
시각화데이터 시각화
자료를 그래프·차트로 표현하여 패턴을 쉽게 파악하게 함.
예) 막대그래프, 산점도, 누적 막대그래프

1
막대그래프 — 국가별 총인구 비교
항목별 크기를 비교할 때 사용
01상단 [Graph] 버튼 클릭 → 빈 그래프 생성
02테이블의 국가명 열을 그래프 x축으로 드래그
03테이블의 인구수(천만명) 열을 그래프 y축으로 드래그
04그래프 오른쪽 [Configuration] → Color를 지역으로 설정하면 지역별 색상 구분

2
산점도 — 기대수명 vs 합계출산율
두 변수 사이의 관계(상관관계)를 볼 때 사용
01[Graph] 클릭 → 새 그래프 생성
02기대수명(세)을 x축으로 드래그
03합계출산율(명)을 y축으로 드래그 → 점 12개가 찍힘
04[Configuration] → Point Labels → 국가명 선택 → 나라 이름 표시
05CODAP 핵심 기능: 산점도의 점 하나를 클릭 → 테이블에서 해당 행이 동시에 강조됨!
CODAP 동적 연결(Dynamic Linking) 체험하기
산점도에서 한국 점을 클릭해보세요 → 테이블에서 한국 행이 함께 강조됩니다.
지역별 드래그 선택: 동아시아 국가들을 드래그로 선택하면 모든 그래프에서 동시에 강조됩니다.

인구구조 비교 — 국가별 노년·유소년 비율
CODAP에서 여러 속성을 비교하는 방법
01[Graph] 클릭 → 새 그래프 생성
02국가명을 x축으로 드래그
03노년비율(%)을 y축으로 드래그 → 막대그래프 생성

05동적 연결 활용: 노년비율 그래프에서 일본 클릭 → 유소년비율 그래프에서도 일본이 강조됨
